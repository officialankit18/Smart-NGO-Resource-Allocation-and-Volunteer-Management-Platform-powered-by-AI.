# NGOHubX

Transforming community needs into verified action through smart technology.

NGOHubX is an AI-powered resource allocation and volunteer management platform built to connect NGOs, volunteers, and citizens in one unified ecosystem. The platform helps organizations identify urgent needs, assign the right volunteers, track progress in real time, and deliver faster social impact.

## Why NGOHubX?

Many NGOs still depend on paper surveys, spreadsheets, calls, and manual coordination. This leads to delayed response, poor visibility, and inefficient resource distribution.

NGOHubX solves these challenges with a centralized digital system designed for speed, transparency, and scale.

## Key Highlights

### NGO Admin Portal
- Smart dashboard with analytics
- Create and assign tasks
- Manage volunteers
- Verify citizen reports
- Monitor task progress
- Generate impact insights

### Volunteer Portal
- Personalized dashboard
- View assigned and nearby tasks
- Accept and complete tasks
- Upload proof of completion
- Earn ratings and badges
- Track contribution history

### Citizen Portal
- Report urgent issues instantly
- Add location and proof images
- Set urgency level
- Track report resolution status

## Google AI Integration

NGOHubX uses Google Gemini AI for intelligent decision-making.

### AI Features
- Detect urgency level from reports
- Categorize issues automatically
- Generate concise summaries
- Suggest recommended actions
- Improve task prioritization

Citizen reports food shortage in Sector 8
